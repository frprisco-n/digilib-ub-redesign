📚 Book Cover Downloader

📖 Panduan Lengkap

Langkah 1: Klik salah satu tombol link (Goodreads, Google Books, atau Amazon) di bawah untuk membuka halaman buku
Langkah 2: Cari gambar cover di halaman tersebut
Langkah 3: Klik kanan pada gambar cover → "Save image as..." (atau "Simpan gambar sebagai...")
Langkah 4: Navigasi ke folder: src/imports/covers/
Langkah 5: Simpan dengan nama: book_[ID].jpg (contoh: book_17.jpg)
Langkah 6: Setelah semua cover tersimpan, update BookCollectionPage.tsx dengan import statements baru

💻 Script Import (copy ke BookCollectionPage.tsx)
